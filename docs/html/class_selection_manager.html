◆ CheckMouseInput()

void SelectionManager.CheckMouseInput ( )
inlineprivate

Checks for mouse input to select/deselect objects every frame, if left mouse button is clicked it shoots a ray and checks if the object that is hit has a GameComponent script or not, if yes then it select else it deselects. We use EventSystem.current.IsPointerOverGameObject() to check if the mouse is over a UI element, if yes then we don't deselect the game object.
